# NFT validator UI

This project was made for HackZurich 2022 / SBHack, Velas challenge #5.2/2.2.
This is a demonstration of a NFT validator.

Demo: https://n0lab.com/projects/hackzurich2022/

## Available Scripts

`npm start`
`npm run build`
